## Contexto
Sistemas de datos para operaciones y análisis, combinando herramientas relacionales y NoSQL según necesidad.

## Problema
Sin una estructura clara, el dato se vuelve difícil de consultar, auditar y convertir en decisiones (KPIs, reportes, análisis causal).

## Solución
- Modelos de datos orientados a trazabilidad y reporting.
- Normalización donde aplica y colecciones NoSQL donde aporta flexibilidad.
- Scripts para automatización de tareas repetitivas.

## Resultados
- Mayor consistencia del dato y mejor calidad de reportes.
- Mejores bases para KPIs y análisis operativo.
